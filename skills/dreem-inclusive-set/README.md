# dreem-inclusive-set

Your current on-model shot, extended across body types and sizes, ages, or genders. Same garment, same styling, same shot family - different people.

## Try it

> "Show this dress on every size - here's our PDP shot."

> "Same look, but across ages - 20s through 60s."

> "This hoodie is unisex. Show it on men and women, same vibe."

## Good to know

It works from the on-model image you already approved: the garment and styling carry over, only the person changes. Pick one axis per set - body/size, age, or gender (combine on request; the cost preview shows the multiplication). Size words map to Dreem's Body Build terms - say "every size" and it casts Slim through Plus, say "XL" and it names the closest build in the plan. Styling pieces pin best when they are your own products (it pulls them from your store). 4 talents x 2 shots is 8 generations - the math is on the table before anything runs.

## Needs

The Dreem connector + the product's current on-model image (PDP URL, upload, or an earlier pack output).
