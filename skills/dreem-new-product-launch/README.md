# dreem-new-product-launch

Launch-day imagery, planned backwards from the live date. One product or a whole drop: PDP on-model set, marketplace stills, social formats, and video - one plan, one cost preview, one approval.

## Try it

> "Three new products go live Thursday - get them launch-ready. Photos attached."

> "New drop next week: plan the imagery for these 5, PDP first."

## Good to know

The set keeps one look across the whole drop - one talent, one styling, one shot style - so it reads as a collection. Tight deadline? It cuts to PDP + marketplace first and ships social and video as a second wave. Video always chains from the drop's own on-model outputs, never from raw photos.

## Needs

The Dreem connector + product photos (front; add backs to unlock back views) + the live date.
