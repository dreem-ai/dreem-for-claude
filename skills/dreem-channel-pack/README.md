# dreem-channel-pack

One finished shot, every channel: PDP, marketplace 1:1, Stories 9:16, feed 4:5, email hero, Pinterest - re-generated at each ratio, never cropped.

## Try it

> "Make this shot work everywhere - feed, Stories, email, and Amazon."

> "Take the winner from that last run and give me the full channel set."

## Good to know

Cropping amputates a composition; this recomposes the shot for each ratio instead. Shots generated by this pack keep the exact same model (the talent is reused); a store photo of a real person gets re-cast with a Dreem talent - it says so before planning. You approve the channel list and cost first.

## Needs

The Dreem connector + one shot you like (a manifest row from another skill, a store image URL, or an upload).
