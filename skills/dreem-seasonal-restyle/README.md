# dreem-seasonal-restyle

Retheme existing on-model imagery for a season or campaign in one batch - same garments, same store format, new styling and mood.

## Try it

> "Our store still looks like summer - restyle these 8 products for fall."

> "Holiday looks for the knitwear collection, cozy and warm."

## Good to know

One seasonal look is locked for the whole batch - shot styles, casting, and styling pieces (ideally from your own catalog, which makes the imagery cross-sell) - so the set reads as one campaign. Garments stay exactly your garments. Products with no on-model shot route through dreem-virtual-model-shots first.

## Needs

The Dreem connector + the products' current on-model images (store URL or uploads) + a season or theme.
